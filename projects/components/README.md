# FrameUI Components

Accessible Angular UI primitives for FrameUI.

## Install

```bash
npm install @frame-ui-ng/components @frame-ui-ng/foundation
```

## Styles

Import the bundled FrameUI styles once in your global stylesheet:

```css
@import '@frame-ui-ng/components/styles.css';
```

This includes the foundation token styles and all component styles.

## Usage

```ts
import { FrButton } from '@frame-ui-ng/components/button';
```

```html
<button frButton>Save changes</button>
```

Documentation: https://frame-ui.com
